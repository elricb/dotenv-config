# dotenv-config

Project `.env` loader.

## Install

```
$ npm install --save-dev @elricb/dotenv-config
```

## Usage

Include and call with base directory.  Searches up from base directory for `.env`.

```js
import dotenv from "@elricb/dotenv-config";
dotenv(__dirname);
```
